Grid

Creating a Grid

You define a container and then use the display property to say this container is a grid like this:

.container {
display: grid;
}

Then you indicate how many rows and columns you want in the container by using grid-template

You also use grid-template to indicate how big you want these rows and columns to be

By default a grid container has one columns

Grid-template

grid-template: grid-template-rows / grid-template-columns;

grid-template-rows

#template-rows {
grid-template-rows: repeat(5, 45px);

Grid Items

The individual blogs in a grid

We can make these blocks span several rows and/or columns

Row grid lines and column grid lines start at 1 and end at a value that is 1 greater than the number of rows or columns the grid has.

grid-row: grid-row-start / grid-row-end;

grid-column: grid-column-start / grid-column-end;

grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end;

.d {
grid-row-start: 2;
grid-row-end: 4;
grid-column-start: 1;
grid-column-end: 3;
}

.e { grid-row-start: 2;
grid-row-end: 4;
}

.b { grid-column-start: 2;
grid-column-end: 4;
}

Grid-template-areas

It is declared in the container.

#grid-template-areas {
grid-template-areas: "heading1 heading1" "main1 main2" "footer1 footer1";
grid-template-rows: 1fr 2fr 1fr;
grid-template-columns: 2fr 1fr;
gap: 5px;
}

grid-area: heading1;
}

grid-area: main1;
