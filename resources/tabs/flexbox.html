Flexbox

Important definitions

One of the most important concepts to grasp in Flexbox are flex containers and flex items. It's important to know the difference between the two because some properties refer to flex containers while others refer to flex items.


Inline-flex

It is used when you want an element's children to be flex items but have the element behave as an inline element

Main-axis

It it is the axis running in the direction the flex items are laid out. For example, as rows across a page or columns down a page.

Cross-axis

It is the axis running perpendicular to the direction the flex items are laid out

Flex container

  • An element on a page that contains flex items
  • You make a container a flex container by setting its display property to flex
  • Properties such as justify-content, align-items, flex grow, flex shrink, etc specify how its children will behave

Flex items

All direct child elements of a flex container

Flex

  • A unitless proportion value thatt dictates how much available space along the main axis each flex item will take compared to other flex items
  • You apply this property to flex items
  • It is also a shorthand property for flex-grow, flex-shrink and flex-basis
  • flex: flex-grow/flex-shrink flex-basis;
  • The flex-basis value is optional
  • Margins are unaffected by flex-grow and flex-shrink

Flex-grow

This property specifies how much of the remaining space in the flex container should be assigned to the item (the flex grow factor).

The default is zero

Cannot grow beyond the max-width property

Flex-shrink

If the size of all flex items is larger than the flex container, items shrink to fit according to flex-shrink.

Default value is 1

Flex-basis

The minimum value for a flex item

Flex-flow

Flex-direction and flex-wrap shorthand

flex-flow: flex-direction flex-wrap;

Flex-direction

It sets out the direction the main axxis runs (which direction the flexbox children are laid out)

Default is row

Flex-wrap

Overflow is moved to the next line

Justify-Content

  • It specifies how flex items spread out from left to right along the main axis of the flex container
  • The container is given a display property of flex and a justify-content property
  • The values can be flex-start, flex-end, center, space-around, space-between and space-evenly

Align-Items

  • It is used to align flex items vertically
  • It tells items how to align along the cross-axis of the parent container
  • The cross-axis is from the top to bottom of the container
  • It's values are flex-start, flex-end, center, baseline and stretch
